# Internet-Speed-Tester-Python
A desktop application built with Python (Tkinter) to measure Internet download and upload speeds using the Speedtest-CLI library.


# 🚀 Python Internet Speed Tester (GUI)

A simple and user-friendly application that tests your internet download and upload speed. It's built using Python's Tkinter library and
the speedtest-cli module.

## ✨ Features
* Real-time download speed calculation.
* Real-time upload speed calculation.
* Clean and simple Graphical User Interface (GUI).
* Easy to use for beginners.

## 🛠️ Installation & Setup

1. **Install Python:** Make sure Python is installed in your system.
2. **Install Library:** Run the below command in Terminal:

   ```bash
   pip install speedtest-cli
